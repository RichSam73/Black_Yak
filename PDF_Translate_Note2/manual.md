# PDF Translator 사용자 매뉴얼

## 서비스 개요

**PDF Translator**는 의류 기술 문서(Tech Pack)를 자동으로 번역하는 웹 서비스입니다.

- **서비스 URL**: https://pdf-translator-970612911654.asia-northeast3.run.app
- **지원 언어**: 영어, 베트남어, 중국어, 인도네시아어, 벵골어

---

## 1. 로그인

1. 서비스 URL에 접속합니다
2. 아이디와 비밀번호를 입력합니다
3. **로그인** 버튼을 클릭합니다

| 항목 | 값 |
|------|-----|
| 아이디 | sije_sam |
| 비밀번호 | sije_sam |

---

## 2. PDF 번역하기

### 2.1 파일 업로드

1. **파일 선택** 버튼을 클릭합니다
2. 번역할 PDF 파일을 선택합니다
3. 파일이 업로드되면 미리보기가 표시됩니다

### 2.2 번역 설정

| 설정 | 설명 | 권장값 |
|------|------|--------|
| **대상 언어** | 번역할 언어 선택 | English |
| **AI 엔진** | 번역에 사용할 AI | Gemini 2.0 Flash |
| **OCR 엔진** | 텍스트 인식 엔진 | Google Vision |

### 2.3 번역 실행

1. **번역 시작** 버튼을 클릭합니다
2. 진행 상황이 표시됩니다:
   - OCR 처리 중...
   - 번역 중...
3. 완료되면 번역된 이미지가 표시됩니다

---

## 3. 번역 결과 확인

### 3.1 미리보기

- 번역된 각 페이지를 미리볼 수 있습니다
- **페이지 탐색**: 좌우 화살표로 페이지 이동
- **확대/축소**: Ctrl + 마우스 휠

### 3.2 텍스트 수정

번역이 마음에 들지 않는 경우:

1. 수정할 텍스트 박스를 클릭합니다
2. 새 번역을 입력합니다
3. **확정** 버튼을 클릭합니다

---

## 4. 메모 추가 (선택사항)

번역된 이미지에 메모를 추가할 수 있습니다.

### 4.1 메모 추가 방법

1. 이미지에서 **우클릭**합니다
2. **메모 추가**를 선택합니다
3. 메모 내용을 입력합니다
4. 스타일을 설정합니다:
   - 글꼴 크기
   - 글꼴 색상
   - 배경색
   - 테두리

### 4.2 화살표 추가

1. 메모 박스 가장자리를 **우클릭 드래그**합니다
2. 연결할 위치까지 드래그합니다
3. 화살표가 자동으로 생성됩니다

---

## 5. 결과 다운로드

### 5.1 개별 다운로드

1. 각 페이지 하단의 **다운로드** 버튼 클릭
2. PNG 이미지로 저장됩니다

### 5.2 전체 다운로드

1. **전체 다운로드** 버튼 클릭
2. 모든 페이지가 ZIP 파일로 다운로드됩니다

---

## 6. 용어 사전 관리

자주 사용하는 의류 용어를 사전에 등록하면 일관된 번역을 받을 수 있습니다.

### 6.1 사전 열기

1. 상단의 **📖** (사전) 버튼을 클릭합니다

### 6.2 용어 추가

**방법 1: 직접 입력**
1. 한글 용어 입력
2. 영어 번역 입력
3. **추가** 버튼 클릭

**방법 2: AI 자동 추가**
1. 자연어로 입력: "겉감을 영어로 shell fabric으로 추가해줘"
2. AI가 자동으로 파싱하여 사전에 추가

### 6.3 용어 수정/삭제

- **수정**: 용어를 클릭하여 편집
- **삭제**: 용어 옆 **X** 버튼 클릭

---

## 7. 로그아웃

1. 우측 상단의 **로그아웃** 버튼 클릭
2. 로그인 페이지로 이동합니다

- **원인**: API 연결 문제

- **원인**: 페이지 수가 많거나 서버 부하
- **해결**: 잠시 후 다시 시도하세요
